# AI-Based-Text-Question-Answering-Bot
Project
https://ai-based-text-question-answering-bot-02586.streamlit.app/

📌 Project Overview
An AI-powered Question Answering Chatbot built using Natural Language Processing (NLP) techniques. The system retrieves accurate answers from large text datasets based on user queries using TF-IDF Vectorization and Cosine Similarity.

🔹 Features
Answers user queries related to students, faculty, events & technical topics
Custom real-world dataset containing academic & institutional information
Interactive Streamlit web interface for real-time Q&A

🛠️ Tech Stack
* Language: Python
* NLP: NLTK, Scikit-learn
* Techniques: TF-IDF Vectorization, Cosine Similarity
* Deployment: Streamlit

🔹 How It Works
User inputs a question
System vectorizes query using TF-IDF
Cosine Similarity finds the most relevant answer from dataset
Best matching answer is returned to the user

📂 Dataset
Custom dataset designed with academic, event and technical information relevant to students and faculty queries.
